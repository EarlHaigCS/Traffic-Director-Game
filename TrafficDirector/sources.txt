Libraries Used:

Pygame
PyObjC and dependencies  - to get Pygame working on OS X
PyYaml - to parse yaml files. (http://pyyaml.org)
Py2app - to export the game as an executable file for OS X.
Py2exe - to export the game as an executable file for Windows.


Code Pieces Taken from Other Open Source Projects:

Menu - to create the menus of the game. (https://github.com/BradStevenson/Pygame-Menu)
